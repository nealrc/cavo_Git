DEMO TEST

Each of the following tests contains 4 difficulty levels. The demo tests have been designed the same way as our standard tests, which have the exact same look and feel. However, unlike the standard vocabulary tests, they do NOT monitor your test data, and thus do NOT smartly adjust test difficulties. To enjoy more features of our vocabulary tests, Join Now!

In the demo tests, there are 20 questions in each test.

Good luck!

CHINESE VOCABULARY

In this test, each question will contain one Chinese word, you will select the most appropriate Chinese interpretation for the question.

ENGLISH DICTIONARY

In this test, each question will contain one Chinese word, you will select the most appropriate English interpretation for the question.

PIN YIN TEST

In this test, each question will contain one Chinese word, you will select the most appropriate Pin Yin interpretation for the question.
